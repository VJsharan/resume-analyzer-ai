🧱 FREE & BEGINNER-FRIENDLY TECH STACK (Highly Recommended)

🔹 Frontend (Clean + Professional Look)

Option 1 (Easiest & Modern):

React.js

Tailwind CSS

ShadCN UI (prebuilt classy components)

Recharts (charts)


💡 Why?
Tailwind + ShadCN makes your UI look startup-level with minimal CSS pain.


---

🔹 Backend (Simple & Powerful)

Python

FastAPI (lighter & faster than Flask)

Uvicorn (server)


💡 Why?
FastAPI auto-generates docs → easy demo + viva explanation.


---

🔹 AI / ML (Beginner → Intermediate)

spaCy (resume & JD parsing)

scikit-learn

TF-IDF

Cosine Similarity


SentenceTransformers (MiniLM) (optional upgrade)


💡 Start with TF-IDF → upgrade to embeddings if time allows.


---

🔹 Resume Parsing

PyMuPDF or pdfplumber

python-docx (for .docx resumes)



---

🔹 Database (Free & Simple)

SQLite (local)

OR MongoDB Atlas (Free Tier)



---

🔹 Auth & Storage

Clerk / Firebase Auth (free tier) (optional)

OR skip auth (college-safe)



---

🔹 Deployment (Optional but Bonus)

Frontend → Vercel / Netlify

Backend → Render / Railway (free tier)

4️⃣ Explainable AI (Very Important)

For every recommendation, show:

---

6️⃣ Learning Roadmap (Looks Premium)

Generate:

Number of missing skills

User availability

🧠 UNIQUE TOUCH (Easy but Impressive)

Examiners LOVE this logic.
